# Description
LigaLeague is a Full-Stack Soccer Themed Formation Visualizer. The application architecture consists of a ReactJS frontend that leverages Asynchronous State Management. The frontend communicates via REST APIs to the backend, which is handled by Spring Boot. The backend performs CRUD operations to persist data in PostgreSQL. This project utilizes real-world soccer data sourced from Kaggle. Users are allowed to search up players in the Bundesliga League and access their players statistics. Then, the players can be added and removed from a team formation page.
# Languages, Frameworks, Sources Used

- ReactJS: Frontend
- PostgreSQL: Database
- Kaggle: Data
